About this website

Unplanned Obsolescence is a blog from Alex Petros. It's his second blog, because his first blog didn't feel like the right place to rant about software lifespans, open source development, and HTML.

Professionally, Alex is a core maintainer for htmx and software engineer based in New York City. He is currently freelancing, building software infrastructure for non-software companies, developing a set of practices for building software infrastructure that lasts a really long time with very little maintenance.

At some point he would like to do the same for municipal, state, and federal governments. {% endblock content %}
