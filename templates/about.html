About this website

Unplanned Obsolescence is a blog from me, Alexander Petros.

Professionally, I am a core maintainer for htmx, author of the Triptych Proposals for HTML, and software engineer based in New York City. I build software infrastructure for places where the software is not the point.

I am constantly refining a set of practices for building software infrastructure that lasts a really long time with very little maintenance; I believe it is possible to build a web service that lasts for a hundred years. At some point I would like to do this for municipal, state, and federal governments.

I also did the Recurse Center, a software retreat with a lot of smart people that I highly recommend.

A brief note about Generative AI

Generative AI plays no role whatsoever in this blog, nor, in fact, anything that I create. I do not write, edit, or code with it. There are many reasons for this, but chief among them is that I believe integrating this technology into my process would produce a worse product.

Whether you are exhausted or excited by the possibilities of Generative AI, rest assured, you will not find it here. {% endblock content %}
